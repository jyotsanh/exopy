## Getting Started

To get started with EyeMate development, follow these steps:

1. Clone the repository:

```
git clone https://github.com/Team-Exopy/Agent-Dashboard.git
```

2. Install dependencies for both backend and frontend:

```
cd Agent-Dashboard/backend
```

```
uv sync
```

```
cd ../frontend
```

```
npm install
```

3. Start the backend server:

```
cd ../frontend
npm start
```

5. Access Agent-Dashboard in your browser at `http://localhost:5173`.
