## Getting Started

To get started with Agent-Backend development, follow these steps:

1. Clone the repository:

```
git clone https://github.com/Team-Exopy/Agent-Ai-Service.git
```

2. Install dependencies for backend:

```
cd Agent-Ai-Service
uv sync
```

3. Start the backend server:

```
uv run dev # starts server in localhost
uv run server # starts server in network 0.0.0.0
```

5. Access Agent-Backend Swagger UI in your browser at `http://localhost:8000/docs`.

### 🐳 Running with Docker

Alternatively, you can run the project using a Docker container:

1. **Start the container:**
   ```bash
   sudo docker compose up -d
   ```
2. **Verify the container status:**
    ```bash
    sudo docker ps
    ```
3. **Access the Application: The container port 8000 is mapped to the host machine port 8015.**
    ```bash
    Access Swagger UI at: http://localhost:8015/docs
    ```
